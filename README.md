# Python-Orientado a obejtos
Esse repositório serve para armazenar os códigos que irei fazer 
para aprender orientação a objetos em python. Os códigos feitos e 
e conteúdo utilizado é do vídeo [Python Object Oriented Programming 
(OOP) - Full Course for Beginners](https://www.youtube.com/watch?v=iLRZi0Gu8Go) 
da freeCodeCamp.org com algmas alterações e adaptações. Para a compreensão de 
alguns conceitos o agente de IA Claude será utilizado para um explicação melhor.

# Estrutura de pastas
Cada conceito aprendido será divido em pastas separadas com um arquivo 
main.py como principal e outros arquivos(se necessário) de suporte. Usando
como base os capítulos do vídeo em questão.

# Tecnologias utilizadas

**PyCharm** como IDE.  
**Python 3.13**  
**Git** para versionamento.  
**Git hub** para armazenamento em nuvem.  
**HTTPS** como protocolo de  transferência.
